# Automation Flow Guide

`*.arsflow` lets you orchestrate repeatable engineering steps over a project.

Supported actions:
- `apply_recipe`
- `apply_patch`
- `gate_project`
- `dashboard_project`
- `snapshot_project`
- `create_delivery_dossier`
- `export_integration_package`

CLI:
```bash
apex_robotics_studio export-demo-flow demo_flow.arsflow
apex_robotics_studio run-flow demo_flow.arsflow
```
