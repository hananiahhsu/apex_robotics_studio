# Delivery Dossier Guide

ApexRoboticsStudio v0.9 adds a delivery dossier command that gathers the main engineering artifacts into a handoff folder.

Typical flow:

```bash
apex_robotics_studio create-delivery-dossier demo_workcell.arsproject dossier runtime.ini
```

Artifacts include the copied project, runtime config, dashboard, release gate report, session outputs, and a manifest/index page.
