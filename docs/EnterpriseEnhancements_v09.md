# ApexRoboticsStudio v0.9 Enterprise Enhancements

This release extends the workstation platform in three practical directions:

1. **Project templates**
   - reusable `.arstemplate` files
   - instantiate governed starter projects instead of copying older jobs manually

2. **Regression analysis**
   - compare baseline and candidate projects
   - review gate status, collisions, findings, duration, and path length in one HTML artifact

3. **Delivery dossier**
   - gather project input, runtime config, dashboard, release gate, job report, SVG layout, and execution session into one folder
   - produce `delivery_manifest.json` and `index.html` for handoff
