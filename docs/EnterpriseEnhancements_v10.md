# ApexRoboticsStudio v1.0 Enterprise Enhancements

This release pushes the project from a strong workstation prototype toward a governed delivery platform.

## New capabilities

### 1. Governance approvals
- `.arsapproval` workflow documents
- role-based sign-off
- current fingerprint verification
- HTML approval report

### 2. Snapshot and rollback
- point-in-time snapshot directory
- manifest and HTML index
- project/runtime/approval restore support

### 3. Integration package export
- JSON + CSV machine-readable bundle
- gate and approval state embedded in summary
- segment, finding, and obstacle export

## Interview positioning

You can now present the system as a robot workstation platform that covers:

- authoring
- simulation
- review
- approval
- rollback
- downstream integration

That is much closer to how real industrial software is evaluated inside engineering organizations.
